# p6n.dev — Personal Blog

This repository contains the source code for my blog at  
**https://blog.p6n.dev**

The blog focuses on software engineering, infrastructure,
distributed systems, and other technical topics I explore and document.

## Features

- Minimal, fast, privacy‑respecting design
- Markdown‑based posts
- Static site generation
- Easy deployment workflow

## Development

Clone the repository and install dependencies:

```bash
git clone https://github.com/<your-username>/<your-blog-repo>.git
cd <your-blog-repo>
```

Start the local development server:

```bash
pnpm install
pnpm run dev
```

Build for production:

```bash
pnpm run build
```

## Deployment

The site is deployed automatically on pushes to the main branch using your
preferred host (e.g., Vercel, Netlify, GitHub Pages).
